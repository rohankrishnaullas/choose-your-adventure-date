# Choose‑Your‑Adventure Date

A tiny mobile-first suspense app:
- Pickup (Nizampet) at 2 or 3
- Lunch vibe
- 2 places in Hyderabad
- Late drop (~11-ish) is OK

## GIF placeholders
Edit the `GIFS` object in `index.html` and paste URLs later.

## Run locally

```bash
cd choose-your-adventure-date
python -m http.server 5174
```

Open:
- http://localhost:5174/

For phone testing on the same Wi‑Fi, use your PC LAN IP, e.g.:
- http://192.168.x.x:5174/
